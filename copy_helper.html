Follow these steps to deploy your AI-First Beta Hub backend:

  1. Go to Extensions > Apps Script in your Google Sheet.
  2. Delete any existing code and paste the code below.
  3. Update WHATSAPP_TOKEN and WHATSAPP_PHONE_ID in CONFIG.
  4. Click Deploy > New Deployment.
  5. Set "Execute as" to "Me" and "Who has access" to "Anyone".
